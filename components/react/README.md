# React Components

This is a collection of React components for SavvyCal Appointments. These components are designed to to be copied in to your codebase and used as a starting point for building your own scheduling experiences.

## Development

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
cd components/react
pnpm dev
```

To configure the browser to open when the development server starts, you can set the `BROWSER` environment variable (via the `.env.local` file).

```bash
# .env.local
BROWSER=firefox
```
